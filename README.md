# DEX Arbitrage Monitor

DEX Arbitrage Monitor - це потужний інструмент для моніторингу та аналізу арбітражних можливостей на децентралізованих біржах (DEX).

## Особливості

- Моніторинг цін на 4 DEX: Uniswap V3, PancakeSwap V3, SushiSwap, Curve
- Відстеження 40 криптовалютних токенів
- Автоматичне виявлення арбітражних можливостей
- Налаштування індивідуальних порогів для кожного токена
- Темна та світла теми інтерфейсу
- **CoinGecko API** для отримання актуальних цін криптовалют
- Зручний інтерфейс на базі PySide6 (Qt 6)

## Технології

- **API**: CoinGecko Free API 
- **Кеш**: 30-секундний TTL для оптимізації запитів
- **Симуляція DEX**: Генерація варіацій цін (±0.1-2%) для різних бірж
- **GUI**: PySide6 з підтримкою темної/світлої тем
- **База даних**: SQLite для зберігання токенів та налаштувань

## Використання

### Основні вкладки

1. **Available Tokens** - Доступні токени для відстеження
2. **Tracked Tokens** - Токени, які відстежуються
3. **Current Prices** - Поточні ціни на DEX
4. **Arbitrage Opportunities** - Знайдені арбітражні можливості

### Робота з токенами

- **Додати токен**: Клікніть на будь-яку клітинку рядка у вкладці "Available Tokens" для вибору, потім натисніть "Add Selected to Tracked"
- **Видалити токен**: У вкладці "Tracked Tokens" натисніть кнопку "Remove" біля потрібного токена
- **Редагувати поріг**: Подвійний клік на значення порога у вкладці "Tracked Tokens"

### Оновлення цін

- Натисніть **Ctrl+R** або кнопку "Refresh Prices" у панелі інструментів
- Одним запитом оновлюються ціни на всіх 3 вкладках (Available Tokens, Tracked Tokens, Current Prices)

### Сканування арбітражу

1. Перейдіть у вкладку "Arbitrage Opportunities"
2. Натисніть **Ctrl+S** або "Scan for Opportunities"
3. Зачекайте завершення сканування

## Конфігурація

Редагуйте `config.py` для налаштування:

- **Токени**: Список з 40 найпопулярніших токенів
- **DEX**: Назви 4 популярних бірж (тільки для відображення)
- **Кеш**: TTL для CoinGecko запитів (30 секунд)
- **Пороги**: Стандартні значення для виявлення арбітражу
- **Тема**: Темна або світла (за замовчуванням: темна)

## Як працює симуляція DEX

Оскільки додаток використовує безкоштовний CoinGecko API (без доступу до реальних DEX цін), він симулює варіації цін:

1. **Базова ціна**: Отримується з CoinGecko (реальна ринкова ціна)
2. **Варіація DEX**: Генерується випадкова варіація від -0.1% до +2%
3. **Результат**: Кожна біржа має трохи різну ціну 

Це дозволяє демонструвати роботу арбітражного алгоритму без необхідності платних API.
